# ShooterGame
 A ShootEm Up made in Unity3D.

# Features
 Multiple Ship Level.

 Two differents Weapons.

 3 Levels.

 Power-Ups !

 A Shop.

 Multiple type of Enemies.

 CheckPoint

# Victory Conditions
 Kill all enemies asked per levels or finish every waves.

 Finish all 3 levels and beat the boss.
